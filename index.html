03.Some Things I’ve Built

Featured Project

DorinaServer · Home Lab Media Server

DorinaServer is my home lab media server running TrueNAS SCALE. It manages a dedicated ZFS media pool, Plex/Jellyfin for streaming, and a full Servarr stack (Sonarr/Radarr/Lidarr/Readarr/qBittorrent/Prowlarr) for automated downloads and library organization. This widget surfaces storage, system, and library information in a compact, portfolio‑friendly layout. Beyond being a media server, DorinaServer is my playground for learning storage, networking, monitoring, and automation — the same skills that show up when debugging production systems or supporting other people’s environments.
